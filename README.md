# Task Manager CLI (Go) 🚀

Простой и надежный консольный менеджер задач, написанный на языке Go. Инструмент позволяет управлять списком дел прямо из терминала, сохраняя все данные в локальный JSON-файл.

## ✨ Основные возможности
- **Добавление задач:** Быстрое создание новых записей.
- **Статус выполнения:** Возможность отмечать задачи как выполненные (чекбокс `[X]`).
- **Удаление:** Очистка списка от ненужных или ошибочных записей.
- **Локальное хранение:** Все данные сохраняются в `tasks.json` в текущей директории.
- **Нулевые зависимости:** Программа использует только стандартную библиотеку Go.

## 🛠 Установка и сборка

1. Склонируйте репозиторий:
   ```bash
   git clone [https://github.com/Corproal0/CLI-utility.git](https://github.com/Corproal0/CLI-utility.git)
   cd CLI-utility
   Скомпилируйте исполняемый файл:

  Bash
  go build -o todo.exe
  Управление программой осуществляется через флаги в командной строке:
  Команда,Описание,Пример
  -add,Добавить новую задачу,".\todo.exe -add ""Изучить интерфейсы в Go"""
  -list,Показать список всех задач,.\todo.exe -list
  -done,Отметить задачу как выполненную (по ID),.\todo.exe -done 1
  -del,Удалить задачу из списка (по ID),.\todo.exe -del 1
